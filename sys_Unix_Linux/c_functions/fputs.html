fputs – вывод строки в указанный поток данных.

Синтаксис:

#include < stdio.h >
int fputs (const char *str, FILE *stream);

Аргументы:

str – указатель на выводимую строку.
stream – указатель на поток данных.

Возвращаемое значение:

В случае успешного вывода слова возвращается неотрицательная величина.

Если при выводе строки произошла ошибка, то возвращается EOF (обычно -1).

Описание:

Функция fputs() выводит строку, на которую указывает аргумент str, в поток данных, на который указывает аргумент stream.

Пример:

В примере выводится строка «write test» и перевод строки с помощью функции fputs в стандартный поток вывода (stdout). 


#include < stdio.h >  //Для fputs

int main (void)
{  
   // Вывод строки «write test» в стандартный 
   // поток вывода
   fputs (“write test\n”,stdout);

   return 0;
}

Результат:

write test
